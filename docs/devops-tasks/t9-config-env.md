- 工数目安:

## 着手条件

- [ ] 現行の設定読み込み箇所（通貨ペア・時間足・period）を特定できている。
- [ ] 環境変数名とデフォルト値の方針が合意されている。
- [ ] 既存フローに影響する破壊的変更（未設定時の挙動変更）がない設計になっている。

## Done定義

- [ ] 通貨ペア・時間足・period が環境変数で上書き可能。
- [ ] 環境変数未設定時は既存デフォルト値で従来どおり動作する。
- [ ] 不正値入力時の挙動（エラーまたはフォールバック）が明確で、テストで確認できる。
- [ ] 設定方法がドキュメント化されている。
